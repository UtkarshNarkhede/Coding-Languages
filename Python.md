## 🔹 1. Basic Data Types
### **Numbers**
- **int** → whole numbers (`x = 10`)
- **float** → decimal numbers (`y = 3.14`)
- **complex** → numbers with real & imaginary part (`z = 2+3j`)

**Functions & Notes:**
```python
type(10)         # int
int(3.9)         # 3
float(5)         # 5.0
complex(2, 3)    # (2+3j)
abs(-7)          # 7
pow(2, 3)        # 8
round(3.14159, 2)# 3.14
